БЛОК «ВІДГУКИ КОМПАНІЇ» — пакет для розробника
================================================

Вміст:
  reviews.html — готовий блок: HTML + CSS + невеликий скрипт стрілок в одному файлі

Як вставити (3 кроки):
  1. З файлу reviews.html візьміть блок <style>…</style> і вставте один раз
     у <head> (або перенесіть у спільний CSS).
  2. Блок <section class="fr-reviews">…</section> вставте у потрібне місце
     сторінки. За макетом — НАД блоком «Топ агенти» (порядок: Відгуки → Топ агенти → Забудовники).
  3. Скрипт гортання (стрілки) вже всередині <section>, нічого окремо підключати не треба.

Особливості:
  • Без бібліотек — чистий HTML + CSS + кілька рядків JS для стрілок.
  • Шрифт успадковується від сайту (font-family: inherit) — блок у стилі сайту.
  • Заголовок ліворуч — як у «Топ агенти» та «Забудовники».
  • Адаптив: десктоп — 3 картки в ряд; мобайл — горизонтальна стрічка (свайп) + стрілки.
  • Усі класи з префіксом .fr- — не конфліктують з рештою стилів.

ВІДГУКИ З GOOGLE
  • Значок — офіційний кольоровий логотип Google (видно джерело).
  • Шапка: логотип Google + рейтинг + к-ть відгуків (зараз 4,9 / 127 — замініть на реальні).
  • Відгуки збираються зі сторінок офісів FAKTOR у Google.
  • Тексти, імена, дати, ініціали та колір аватара у картках — ЗАМІНІТЬ на реальні з Google.
    Кожна картка — один блок <article class="fr-card">…</article>; копіюйте/видаляйте за потребою.

  Підказка для авто-підтягування (опційно, якщо робите динамічно):
  відгуки офісів можна тягнути через Google Places API (Place Details → reviews)
  по place_id кожного офісу, або через готовий віджет (напр. сервіси агрегації Google-відгуків).
  Поточний блок — статична верстка-зразок під ваш дизайн.

ЗМІНА РЕЙТИНГУ
  Число «4,9» і «127 відгуків» — у шапці (.fr-rate / .fr-cnt). Зірок завжди 5 (повні).
