БЛОК «ТОП АГЕНТИ FAKTOR» — пакет для розробника
================================================

Вміст:
  top-agents.html  — готовий блок: HTML + CSS в одному файлі
  assets/          — 5 фото агентів (4:5, єдиний світлий фон)

Як вставити (3 кроки):
  1. Скопіюйте папку assets/ у проєкт сайту.
     Якщо шлях до зображень інший — поправте src у тегах <img>.
  2. З файлу top-agents.html візьміть блок <style>…</style>
     і вставте його один раз у <head> (або перенесіть у спільний CSS).
  3. Блок <section class="fa-agents">…</section> вставте у потрібне
     місце сторінки (туди, де має бути секція топ-агентів).

Особливості:
  • Без JS і без бібліотек — чистий HTML + CSS.
  • Шрифт успадковується від сайту (font-family: inherit) — секція
    автоматично виглядає в стилі сайту.
  • Адаптив: десктоп — 5 карток у ряд; планшет — 3; мобайл — свайп-стрічка.
  • Ефекти ховеру (підйом картки + зум фото) — на чистому CSS.
  • Усі класи з префіксом .fa- — не конфліктують з рештою стилів сайту.

Щоб додати/змінити агента — скопіюйте один блок <article class="fa-card">…</article>,
поміняйте фото (src), ім'я, офіс і телефон (текст + href="tel:…").

Якщо агент без фото (заглушка):
  Замість блоку <div class="fa-card__photo"><img…></div> вставте:
    <div class="fa-card__photo fa-card__photo--empty">
      <img src="assets/faktor-logo.svg" alt="FAKTOR">
    </div>
  Показує логотип FAKTOR на нейтральному фоні.
  Готовий приклад є в кінці top-agents.html у коментарі.
  Логотип лежить у assets/faktor-logo.svg (навігаційний, navy).
